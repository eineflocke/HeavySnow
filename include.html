何これ
  • 毎時 20, 50 分ごろに更新します。
  • 高頻度で本体の更新を行っているため、定期的に Ctrl + F5 でページ再読み込みを実施願います。
  • IE11 での使用は一応サポートしますが大変重いため推奨されません。
  • バグ（普段使いの範囲で発現し得るもの）・機能に関するお問い合わせは岡野・余郷まで
  • 主な対象は北陸ですが、下記理由のためその他幾つかの府県も対象としています。
    • 栃木・群馬：本庁2府県担当からの現業利用要望を受けて
    • 青森：知人からの現業利用要望を受けて
    • その他：なんとなく
  • 地点追加には csv 形式の地点データ、データ取得とデコードの Python スクリプトをご用意頂く必要があります。詳しくは余郷まで
使い方
  • 大雪監視ページには報知機能が搭載されています。
    ページ読み込み後に報知スイッチをOnにすると、最新データを取得した際、基準を超過していた場合に報知します。
  • 自動更新ボタンがOnの時、5分毎に自動更新されます。
    更新毎に最新時刻のデータが表示されるため、過去データを閲覧する際は、自動更新ボタンをOffにしてください。
  • プルダウンで、表示地域、機関、要素を変える事が可能です。
  • 「＜」「最新」「＞」ボタンで時刻を変えられます。「＜」は1時間前に、「＞」は1時間後に移動します。
    テキストボックスに時刻を「YYYYMMDDHH」形式で入力して「設定」ボタンもしくはEnterキーを押すと、指定した時刻に移動します。
  • 報知は区分欄が平地、山地等となっている情報掲載対象地点のみを対象としています。
    詳しくは以下を参照
    \\tkvfile2\管内共有\07_新潟地方気象台\大雪ツール
データの見方
  • 降雪深は、1時間毎の積雪深の増分の和（増分がマイナスの場合は無視）を計算して表示します。
    例えば積雪深が10cm→12cm→15cm→13cmと変化したときには、2cm+3cm=5cmとなります（最後の-2cmは無視されます）。
  • 同地点で気温が観測されている場合、4℃以上且つ前1時間積雪深が 0cm のとき S1 を 0F とする。
  • S1 が 22cm 以上のとき疑問値「#」とする。
  • S1 が欠測の場合、前時刻の S1 が
    • 正常値の場合、S1 を 0) 、積雪深を「)」
    • 「)」の場合、S1 を 0] 、積雪深を「]」
    したがって、最大2時間降雪を 0 とみなして表示する。データの欠測・入電遅れが多い部外観測所のデータを参照しやすくするための措置
  • 1時間降雪深の「F」は正常値と同等の扱いとし、S3～24の累積期間中に1時間降雪深の「)」「]」「#」が
    • 40% 超の場合、無効値
    • 20% 超の場合、]
    • 0% 超の場合、)
    • 0%の場合、正常値
  • 積雪深差は、現在積雪深から対象時刻積雪深を差し引いた値となります。降雪深とは異なり値が負になることもあります。
    例えば積雪深が13cm→14cm→12cm→10cmと変化したときには、10cm-13cm=-3cmとなります。
キー操作
  • 初期値時刻入力中に Enter キー：初期値時刻決定
  • 左右キー：時刻を前後１時間に移動
  • Shift + 上下キー：表示地域の変更
  • Alt + 上下キー：観測機関の変更
  • Ctrl + 上下キー：地図表示要素の変更
